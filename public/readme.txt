DemoSite — Analytics Test Fixture
==================================

This site demonstrates common GA4 trackable interactions:

1. Outbound Clicks   → /outbound
2. Form Submission   → /form
3. Video Play        → /video
4. File Download     → /download

Each page logs a console event as a placeholder.
Replace console.log calls with gtag() or dataLayer.push()
to wire up real Google Analytics 4 tracking.

Built with: React + Vite + Tailwind CSS + React Router + React Hook Form + react-player
